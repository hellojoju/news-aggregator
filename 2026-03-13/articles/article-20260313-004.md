# OpenClaw发布v2.0：原生支持MCP，Agent生态全面升级

**分类**: AI编程  
**发布时间**: 2026-03-13  
**来源数量**: 1  

## 正文

OpenClaw今日发布v2.0版本，这是该项目迄今为止最大的更新。新版本原生支持Model Context Protocol（MCP），大幅提升了AI Agent的扩展能力。

**核心更新：**

1. **MCP原生支持**：OpenClaw 2.0内置MCP客户端，可以直接连接任何MCP服务器。这意味着Agent可以无缝使用外部工具，如文件系统、数据库、API等。

2. **Skill市场**：推出官方Skill市场，开发者可以发布和分享自己的Skill。目前已有超过200个社区Skill，涵盖搜索、代码分析、文档生成等场景。

3. **多Agent协作**：支持多个Agent并行工作，通过消息队列协调任务。适合复杂的自动化工作流。

4. **增强的内存管理**：改进的长期记忆系统，支持向量数据库存储和检索。

5. **Web界面**：全新的Web UI，可以可视化查看Agent执行过程、管理任务和配置。

**MCP集成示例：**

```javascript
// 连接MCP服务器
const mcpClient = await openclaw.connectMCP({
  serverUrl: 'http://localhost:3000',
  tools: ['file_system', 'web_search']
});

// Agent使用MCP工具
const agent = openclaw.createAgent({
  tools: [mcpClient.tools]
});
```

**社区反响：**

OpenClaw 2.0发布后，GitHub Star数单日增长超过2000。开发者社区对MCP支持反响热烈，认为这是Agent生态的重要里程碑。

OpenClaw团队表示，v2.0只是一个开始，未来将持续投入Agent基础设施建设。
